# Pertemuan 07 - Scrum Framework dan simulasi Sprint Planning

## Kemampuan Akhir
- SCPMK802-01401: Mahasiswa mampu menjelaskan prinsip dasar manajemen proses RPL dan tahapan SDLC serta mengelola proses pengembangan melalui metodologi preskriptif (Prototyping/Spiral/RUP), metodologi Agile (Scrum/Kanban) dengan manajemen alur kerja visual, penjaminan mutu (SQA) melalui pengujian, dan pemeliharaan/evolusi sistem pasca-rilis.

## Materi Pembelajaran
Scrum Framework dan simulasi Sprint Planning.

## Bentuk dan Metode Pembelajaran
Bentuk: Kuliah dan workshop (simulasi). Metode: Project Based Learning (PBL) fase inisiasi proyek.

## Estimasi Waktu
1 x (4 x 50') tatap muka; tugas terstruktur dan mandiri.

## Pengalaman Belajar Mahasiswa
Mahasiswa melakukan simulasi perencanaan Sprint 1 dalam tim Scrum.

## Penilaian
Kriteria: kesesuaian prosedur events Scrum. Bentuk: penilaian kinerja proyek.

## Indikator Penilaian
Kesesuaian pembagian peran tim; kejelasan definisi Done (DoD).
