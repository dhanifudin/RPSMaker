# Pertemuan 13 - Implementasi (Construction/Coding)

## Kemampuan Akhir
- SCPMK205-01405: Mahasiswa mampu merealisasikan blueprint menjadi komponen aplikasi yang modular, maintainable, dan sesuai standar kode.

## Materi Pembelajaran
Implementasi (Construction/Coding).

## Bentuk dan Metode Pembelajaran
Bentuk: Praktikum lab. Metode: Project Based Learning (tahap konstruksi). Penugasan: pengembangan aplikasi (coding).

## Estimasi Waktu
1 x (4 x 50') tatap muka; tugas terstruktur dan mandiri.

## Pengalaman Belajar Mahasiswa
Mahasiswa melakukan coding secara kolaboratif menggunakan Git dan melakukan code review antar anggota tim.

## Penilaian
Kriteria: aplikasi berjalan sesuai fungsi. Bentuk: source code dan demo aplikasi.

## Indikator Penilaian
Modularitas kode; tidak ada error/bug kritikal; penerapan Clean Code.
