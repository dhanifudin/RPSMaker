# Pertemuan 09 - Konsep Array dan Array 1 Dimensi

## Kemampuan Akhir
- SCPMK704-00604: Mahasiswa mampu menyelesaikan studi kasus menggunakan array 1 dan 2 dimensi serta fungsi iteratif dan rekursif dalam algoritma penyelesaian masalah.

## Materi Pembelajaran
Konsep array; array 1 dimensi; studi kasus searching dan sorting sederhana.

## Bentuk dan Metode Pembelajaran
Modalitas: Daring/Luring. Metode: CTL dan Project Based Learning. Penugasan: Tugas 8 deklarasi dan operasi array 1 dimensi pada kasus searching.

## Estimasi Waktu
1 x 2 x 100' tatap muka; 1 x 2 x 70' tugas terstruktur/mandiri.

## Pengalaman Belajar Mahasiswa
Mahasiswa menjelaskan konsep array 1 dimensi dan mengimplementasikannya untuk kasus searching dan sorting.

## Penilaian
Keaktifan diskusi; ketepatan jawaban tugas. Bentuk: Tugas studi kasus. Mengacu rubrik RTM.

## Indikator Penilaian
Ketepatan konsep array 1 dimensi; kebenaran solusi searching/sorting sederhana.
