# Pertemuan 02 - Konsep Dasar Version Control dan Kanban

## Kemampuan Akhir
- SCPMK903-00601: Mahasiswa mampu menjelaskan konsep dasar algoritma, version control, dan Kanban serta menganalisis permasalahan sederhana ke dalam bentuk algoritma.

## Materi Pembelajaran
Konsep dasar version control; prinsip-prinsip Kanban.

## Bentuk dan Metode Pembelajaran
Modalitas: Daring/Luring. Metode: CTL dan Problem Based Learning. Penugasan: analisis macam-macam version control; membuat Kanban board aktivitas sehari-hari.

## Estimasi Waktu
1 x 2 x 100' tatap muka; 1 x 2 x 70' tugas terstruktur/mandiri.

## Pengalaman Belajar Mahasiswa
Mahasiswa menjelaskan pentingnya version control dan memvisualisasikan aktivitas dengan Kanban board.

## Penilaian
Keaktifan diskusi; ketepatan jawaban tugas. Bentuk: Tugas studi kasus. Mengacu rubrik RTM.

## Indikator Penilaian
Ketepatan konsep version control; kesesuaian Kanban board dengan prinsip visualisasi alur kerja.
