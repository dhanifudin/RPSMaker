# Pertemuan 01 - Instalasi tools pemrograman Java

## Kemampuan Akhir
- SCPMK0903-00701: Mahasiswa mampu menjelaskan konsep program, bahasa pemrograman, compiler, dan debugging, melakukan instalasi tools pemrograman Java, serta memodelkan permasalahan sederhana menjadi algoritma.

## Materi Pembelajaran
- Konsep bahasa pemrograman
- Bahasa pemrograman Java
- Compiler
- Instalasi tools pemrograman Java (JDK, IDE)

## Bentuk dan Metode Pembelajaran
Modalitas: Luring/Daring. Metode: Problem Based Learning. Aktivitas: praktikum jobsheet di laboratorium.

## Estimasi Waktu
1 x 6 x 50' praktikum.

## Pengalaman Belajar Mahasiswa
Mahasiswa melakukan instalasi tools Java sesuai jobsheet, melakukan compile dan debugging sintaks program, serta mengerjakan tugas praktikum (Tugas 1: Jobsheet instalasi Java).

## Penilaian
Bentuk penilaian: jobsheet praktikum dan tugas mandiri. Mengacu rubrik RTM.

## Indikator Penilaian
Ketepatan instalasi tools pemrograman Java; keaktifan diskusi (tanya jawab).
