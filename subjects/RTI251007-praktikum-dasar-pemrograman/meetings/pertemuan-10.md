# Pertemuan 10 - Array 1: array 1 dimensi

## Kemampuan Akhir
- SCPMK0704-00703: Mahasiswa mampu menerapkan array 1 dimensi untuk menyelesaikan studi kasus dalam program Java.

## Materi Pembelajaran
- Array 1 dimensi
- Pengaksesan elemen array
- Pengayaan: searching dan sorting

## Bentuk dan Metode Pembelajaran
Modalitas: Luring/Daring. Metode: Diskusi. Aktivitas: praktikum jobsheet.

## Estimasi Waktu
1 x 6 x 50' praktikum.

## Pengalaman Belajar Mahasiswa
Mahasiswa menerapkan array 1 dimensi menggunakan bahasa Java sesuai jobsheet dan tugas praktikum (Tugas 8: Jobsheet array 1 dimensi).

## Penilaian
Bentuk penilaian: jobsheet praktikum dan tugas mandiri. Mengacu rubrik RTM.

## Indikator Penilaian
Ketepatan menggunakan array 1 dimensi; ketepatan kode program dan output.
