# Pertemuan 05 - Pemilihan 1: if, if-else, else-if, dan switch-case

## Kemampuan Akhir
- SCPMK0704-00702: Mahasiswa mampu menerapkan struktur pemilihan (if, if-else, else-if, dan switch-case) ke dalam program Java.

## Materi Pembelajaran
- Pemilihan if
- Pemilihan if-else
- Pemilihan else-if
- Pemilihan switch-case

## Bentuk dan Metode Pembelajaran
Modalitas: Luring/Daring. Metode: Diskusi dan Problem Based Learning. Aktivitas: praktikum jobsheet.

## Estimasi Waktu
1 x 6 x 50' praktikum.

## Pengalaman Belajar Mahasiswa
Mahasiswa menuliskan program Java dari flowchart kasus pemilihan dasar dan mengerjakan tugas praktikum (Tugas 4: Jobsheet pemilihan 1).

## Penilaian
Bentuk penilaian: jobsheet praktikum dan tugas mandiri. Mengacu rubrik RTM.

## Indikator Penilaian
Ketepatan menggunakan pemilihan 1; ketepatan kode program dan output sesuai tugas.
