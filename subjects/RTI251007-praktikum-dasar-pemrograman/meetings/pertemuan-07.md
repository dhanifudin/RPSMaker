# Pertemuan 07 - Perulangan 1: for, while, do-while

## Kemampuan Akhir
- SCPMK0704-00703: Mahasiswa mampu menerapkan struktur perulangan (for, while, do-while) untuk menyelesaikan studi kasus dalam program Java.

## Materi Pembelajaran
- Perulangan for
- Perulangan while
- Perulangan do-while

## Bentuk dan Metode Pembelajaran
Modalitas: Luring/Daring. Metode: Diskusi. Aktivitas: praktikum jobsheet.

## Estimasi Waktu
1 x 6 x 50' praktikum.

## Pengalaman Belajar Mahasiswa
Mahasiswa menuliskan program Java berdasarkan flowchart kasus perulangan bagian 1 dan memanfaatkan konsep perulangan (Tugas 6: Jobsheet perulangan 1).

## Penilaian
Bentuk penilaian: jobsheet praktikum dan tugas mandiri. Mengacu rubrik RTM.

## Indikator Penilaian
Ketepatan menjelaskan perulangan bagian 1; ketepatan kode program dan output.
