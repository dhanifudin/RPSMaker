# Pertemuan 03 - Input-output, variabel, sequence, dan operator

## Kemampuan Akhir
- SCPMK0704-00702: Mahasiswa mampu menerapkan tipe data, variabel, input-output, sequence, dan operator pada sebuah kode program menggunakan Java.

## Materi Pembelajaran
- Tipe data dan variabel
- Input-output
- Sequence
- Operator: penugasan aritmatika, penugasan gabungan, increment, decrement, relasional, logika, kondisional, bitwise, casting

## Bentuk dan Metode Pembelajaran
Modalitas: Luring/Daring. Metode: Diskusi. Aktivitas: praktikum jobsheet.

## Estimasi Waktu
1 x 6 x 50' praktikum.

## Pengalaman Belajar Mahasiswa
Mahasiswa menggunakan tipe data, variabel, input-output, dan operator serta mentranslasikan permasalahan sequence ke bahasa Java (Tugas 3: Jobsheet input-output, variabel, sequence).

## Penilaian
Bentuk penilaian: jobsheet praktikum dan tugas mandiri. Mengacu rubrik RTM.

## Indikator Penilaian
Ketepatan menggunakan tipe data, variabel, input-output, operator; ketepatan mentranslasikan permasalahan sequence ke Java; ketepatan kode program dan output.
