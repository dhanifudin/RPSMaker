# Pertemuan 16 - PBL: Pengembangan Aplikasi Java GUI+Database

## Kemampuan Akhir
- SCPMK0704-02602: Mahasiswa mampu mengimplementasikan overriding, overloading, interface, kelas abstrak, dan polymorfisme pada kasus nyata.
- SCPMK0903-02603: Mahasiswa mampu membangun aplikasi Java berbasis GUI (Swing/JavaFX) dengan koneksi database JDBC.

## Materi Pembelajaran
PBL: Pengembangan aplikasi Java GUI+Database.

## Bentuk dan Metode Pembelajaran
Modalitas: Blended Learning. Bentuk: Luring/Daring. Metode: Case Method, praktik pemrograman Java, studi kasus, dan peer review.

## Estimasi Waktu
1 x 4 x 50' praktikum; 1 x 1 x 50' tugas/laporan mandiri.

## Pengalaman Belajar Mahasiswa
Mahasiswa mengembangkan aplikasi Java berbasis GUI dan database secara terintegrasi sebagai proyek akhir.

## Penilaian
Bentuk penilaian: PBL Proyek Akhir. Mengacu rubrik RTM.

## Indikator Penilaian
Ketepatan konsep; kualitas implementasi; validasi dan dokumentasi.
