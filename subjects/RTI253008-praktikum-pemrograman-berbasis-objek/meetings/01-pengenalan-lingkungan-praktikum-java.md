# Pertemuan 01 - Pengenalan Lingkungan Praktikum Java dan IDE

## Kemampuan Akhir
- SCPMK210-02601: Mahasiswa mampu mengimplementasikan kelas, enkapsulasi, relasi kelas, dan inheritance dalam kode Java yang berfungsi.

## Materi Pembelajaran
Pengenalan lingkungan praktikum Java dan IDE.

## Bentuk dan Metode Pembelajaran
Modalitas: Blended Learning. Bentuk: Luring/Daring. Metode: Case Method, praktik pemrograman Java, studi kasus, dan peer review.

## Estimasi Waktu
1 x 4 x 50' praktikum; 1 x 1 x 50' tugas/laporan mandiri.

## Pengalaman Belajar Mahasiswa
Mahasiswa mengerjakan jobsheet pengenalan IDE dan membuat program Java pertama, menunjukkan evidence hasil belajar.

## Penilaian
Bentuk penilaian: Tugas Jobsheet. Mengacu rubrik RTM.

## Indikator Penilaian
Ketepatan konsep; kualitas implementasi; validasi dan dokumentasi.
