# Pertemuan 12 - Review unit/integration test; code review; coverage analysis; test maintainability

## Kemampuan Akhir
SCPMK0605-04206: Mahasiswa mampu mengembangkan dan memelihara kualitas kode perangkat lunak sesuai standar melalui unit dan integration testing serta code review.

## Materi Pembelajaran
Review unit/integration test; code review; coverage analysis; test maintainability; standar kualitas kode.

## Bentuk dan Metode Pembelajaran
Modalitas: Blended Learning. Metode: Code Review dan Peer Assessment. Penugasan: Tugas 11 evaluasi test code.

## Estimasi Waktu
1 x 2 x 50' tatap muka; 1 x 2 x 50' tugas kelompok.

## Pengalaman Belajar Mahasiswa
Mahasiswa melakukan review test code, menganalisis coverage, dan memberi rekomendasi perbaikan sesuai standar kualitas kode.

## Penilaian
Bentuk penilaian: Tugas 11. Mengacu Rubrik R13.

## Indikator Penilaian
Ketepatan review; pemahaman coverage; rekomendasi refactoring test; kualitas feedback; kesesuaian standar kualitas kode.
