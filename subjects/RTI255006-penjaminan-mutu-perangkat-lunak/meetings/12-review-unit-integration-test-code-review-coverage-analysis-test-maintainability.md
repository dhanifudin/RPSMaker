# Pertemuan 12 - Review unit/integration test; code review; coverage analysis; test maintainability

## Kemampuan Akhir
SCPMK0608-04203: Mahasiswa mampu menerapkan dan mengevaluasi functional testing menggunakan tools yang sesuai untuk memverifikasi kesesuaian perangkat lunak terhadap requirement.

## Materi Pembelajaran
Review unit/integration test; code review; coverage analysis; test maintainability.

## Bentuk dan Metode Pembelajaran
Modalitas: Blended Learning. Metode: Code Review dan Peer Assessment. Penugasan: Tugas 11 evaluasi test code.

## Estimasi Waktu
1 x 2 x 50' tatap muka; 1 x 2 x 50' tugas kelompok.

## Pengalaman Belajar Mahasiswa
Mahasiswa melakukan review test code, menganalisis coverage, dan memberi rekomendasi perbaikan.

## Penilaian
Bentuk penilaian: Tugas 11. Mengacu Rubrik R13.

## Indikator Penilaian
Ketepatan review; pemahaman coverage; rekomendasi refactoring test; kualitas feedback.
