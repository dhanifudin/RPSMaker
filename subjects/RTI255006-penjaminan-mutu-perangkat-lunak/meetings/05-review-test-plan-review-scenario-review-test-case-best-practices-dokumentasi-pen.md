# Pertemuan 05 - Review test plan; review scenario; review test case; best practices dokumentasi pengujian

## Kemampuan Akhir
SCPMK0608-04202: Mahasiswa mampu merancang dokumen test plan, test scenario, test case, test monitoring, test control, dan test report yang sesuai dengan kebutuhan perangkat lunak.

## Materi Pembelajaran
Review test plan; review scenario; review test case; best practices dokumentasi pengujian.

## Bentuk dan Metode Pembelajaran
Modalitas: Blended Learning. Metode: Peer Review dan Project Based Learning. Penugasan: Tugas 4 review dan presentasi artefak.

## Estimasi Waktu
1 x 2 x 50' tatap muka; 1 x 2 x 50' tugas kelompok.

## Pengalaman Belajar Mahasiswa
Mahasiswa melakukan peer review, mempresentasikan temuan, dan memperbaiki dokumen pengujian.

## Penilaian
Bentuk penilaian: Tugas 4 dan presentasi. Mengacu Rubrik R5.

## Indikator Penilaian
Kedalaman review; rekomendasi; kualitas presentasi; revisi artefak.
