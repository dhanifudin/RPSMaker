# Pertemuan 11 - Unit testing; integration testing; system testing; TDD approach

## Kemampuan Akhir
SCPMK0608-04203: Mahasiswa mampu menerapkan dan mengevaluasi functional testing menggunakan tools yang sesuai untuk memverifikasi kesesuaian perangkat lunak terhadap requirement.

## Materi Pembelajaran
Unit testing; integration testing; system testing; TDD approach.

## Bentuk dan Metode Pembelajaran
Modalitas: Blended Learning. Metode: Coding Practice dan TDD. Penugasan: Tugas 10 unit dan integration test.

## Estimasi Waktu
1 x 2 x 50' tatap muka; 1 x 2 x 50' tugas kelompok.

## Pengalaman Belajar Mahasiswa
Mahasiswa menulis test code, menjalankan test suite, dan mengevaluasi hasil.

## Penilaian
Bentuk penilaian: Tugas 10. Mengacu Rubrik R12.

## Indikator Penilaian
Kualitas test code; coverage fungsi kritis; penerapan TDD; integrasi kode.
