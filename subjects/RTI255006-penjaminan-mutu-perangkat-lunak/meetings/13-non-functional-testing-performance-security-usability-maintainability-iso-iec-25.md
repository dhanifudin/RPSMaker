# Pertemuan 13 - Non-functional testing; performance; security; usability; maintainability; ISO/IEC 25010 quality attributes

## Kemampuan Akhir
- SCPMK0608-04204: Mahasiswa mampu merancang, menerapkan, dan mengevaluasi non-functional testing serta menyusun rekomendasi peningkatan kualitas perangkat lunak berdasarkan quality attributes.
- SCPMK0508-04205: Mahasiswa mampu menerapkan prinsip information assurance dan software security dalam pengujian non-functional dan jaminan mutu perangkat lunak.

## Materi Pembelajaran
Non-functional testing; performance; security; information assurance; software security; usability; maintainability; ISO/IEC 25010 quality attributes.

## Bentuk dan Metode Pembelajaran
Modalitas: Blended Learning. Metode: CTL dan Case Study Analysis. Penugasan: Tugas 12 analisis kebutuhan non-functional testing dan security testing.

## Estimasi Waktu
1 x 2 x 50' tatap muka; 1 x 2 x 50' tugas kelompok.

## Pengalaman Belajar Mahasiswa
Mahasiswa mengidentifikasi quality attributes, merancang strategi non-functional testing, dan menerapkan prinsip information assurance/software security.

## Penilaian
Bentuk penilaian: Tugas 12. Mengacu Rubrik R14.

## Indikator Penilaian
Identifikasi atribut kualitas; relevansi strategi; prioritas risiko; keterkaitan kebutuhan pengguna; penerapan prinsip information assurance dan software security.
