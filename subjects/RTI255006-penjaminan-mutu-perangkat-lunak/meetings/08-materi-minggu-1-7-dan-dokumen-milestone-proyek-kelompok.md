# Pertemuan 08 - Materi minggu 1-7 dan dokumen milestone proyek kelompok

## Kemampuan Akhir
- SCPMK608-04202: Mahasiswa mampu merancang dokumen test plan, test scenario, test case, test monitoring, test control, dan test report yang sesuai dengan kebutuhan perangkat lunak.

## Materi Pembelajaran
Materi minggu 1-7 dan dokumen milestone proyek kelompok.

## Bentuk dan Metode Pembelajaran
Evaluasi: UTS berbasis milestone review PBL kelompok.

## Estimasi Waktu
1 x 3 x 50' evaluasi praktik dan defense proyek.

## Pengalaman Belajar Mahasiswa
Mahasiswa mempresentasikan dan mempertahankan test plan, test scenario, test case, traceability matrix, serta monitoring/control plan proyek.

## Penilaian
Bentuk penilaian: UTS. Mengacu Rubrik R7.

## Indikator Penilaian
Kelengkapan dokumen; konsistensi strategi; traceability; monitoring/control plan; kualitas defense kelompok.
