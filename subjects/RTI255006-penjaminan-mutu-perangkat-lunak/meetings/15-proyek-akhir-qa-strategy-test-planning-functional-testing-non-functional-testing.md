# Pertemuan 15 - Proyek akhir; QA strategy; test planning; functional testing; non-functional testing; reporting; presentation

## Kemampuan Akhir
SCPMK0608-04201-04204, SCPMK0606-04207: Integrasi semua kemampuan dalam proyek komprehensif QA, termasuk penyusunan proses deployment dan rencana pemeliharaan perangkat lunak.

## Materi Pembelajaran
Proyek akhir; QA strategy; test planning; functional testing; non-functional testing; proses deployment; rencana pemeliharaan; reporting; presentation.

## Bentuk dan Metode Pembelajaran
Modalitas: Blended Learning. Metode: Project Based Learning dan mentoring. Penugasan: PBL proyek akhir strategi QA lengkap.

## Estimasi Waktu
2 x 2 x 50' tatap muka; 2 x 2 x 50' tugas kelompok.

## Pengalaman Belajar Mahasiswa
Mahasiswa menyusun strategi QA, mengimplementasikan pengujian, menyusun proses deployment dan rencana pemeliharaan, membuat laporan, dan mempresentasikan hasil.

## Penilaian
Bentuk penilaian: PBL/proyek akhir. Mengacu Rubrik R16.

## Indikator Penilaian
Integrasi strategi; implementasi; evidence pengujian; kelengkapan rencana deployment dan pemeliharaan; dokumentasi; presentasi; rekomendasi mutu.
