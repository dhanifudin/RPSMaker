# Tugas 5: Monitoring, Control, Completion, dan Report

**Bentuk Tugas/Evaluasi:** Project Based Learning

**Sub-CPMK:** SCPMK0608-04202

## Deskripsi
Mahasiswa menyusun monitoring, control, completion, dan laporan pengujian.

## Metode Pengerjaan
Menentukan metrik, memonitor progres uji, mencatat risiko, dan menyusun test report.

## Bentuk Format Luaran
Dokumen test monitoring dan test report.

## Rubrik Penilaian

| Indikator Penilaian | Bobot (%) | Kriteria 3 (Baik) | Kriteria 2 (Cukup) | Kriteria 1 (Kurang) |
| --- | ---: | --- | --- | --- |
| Metrik monitoring dan control | 0.8 | Progress, defect, risk, coverage, dan action item dimonitor dengan metrik yang sesuai. | Metrik tersedia, tetapi belum lengkap atau belum dipakai untuk control. | Monitoring tidak berbasis metrik atau status pengujian tidak jelas. |
| Test completion dan laporan | 0.7 | Completion criteria, summary eksekusi, defect summary, dan status release disusun jelas. | Laporan tersedia, tetapi completion criteria atau defect summary belum lengkap. | Laporan tidak menggambarkan hasil pengujian. |
| Rekomendasi tindak lanjut | 0.5 | Rekomendasi berbasis data monitoring, risiko, dan defect yang ditemukan. | Rekomendasi cukup relevan, tetapi belum kuat berbasis data. | Rekomendasi tidak operasional atau tidak terkait hasil pengujian. |
