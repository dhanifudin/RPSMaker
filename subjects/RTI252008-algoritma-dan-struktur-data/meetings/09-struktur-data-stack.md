# Pertemuan 9 - Struktur Data Stack

## Kemampuan Akhir
- SCPMK0207-01701: Mahasiswa mampu menjelaskan karakteristik berbagai struktur data serta alur pengelolaan data dalam mendukung penyelesaian permasalahan komputasi.

## Materi Pembelajaran
Konsep struktur data stack, operasi stack (push, pop, peek, isEmpty), dan postfix expressions.

## Bentuk dan Metode Pembelajaran
Modalitas: Blended Learning. Bentuk: Luring. Metode: Contextual Teaching and Learning (CTL). Sumber belajar: LMS Polinema.

## Estimasi Waktu
2 x 2 x 100' tatap muka dan tugas terstruktur; 2 x 2 x 70' tugas mandiri; penugasan 1 x 50'.

## Pengalaman Belajar Mahasiswa
Mahasiswa memahami struktur data stack, operasi pada stack, dan mengimplementasikan konsep stack untuk menyelesaikan studi kasus.

## Penilaian
Bentuk penilaian: Tugas Stack. Kriteria: ketepatan dan penguasaan; keaktifan diskusi; ketepatan jawaban tugas minggu 9.

## Indikator Penilaian
Pemahaman penerapan konsep stack; kesesuaian jawaban tugas.
