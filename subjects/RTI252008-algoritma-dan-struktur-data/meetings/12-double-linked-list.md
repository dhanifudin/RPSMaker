# Pertemuan 12 - Double Linked List

## Kemampuan Akhir
- SCPMK0207-01701: Mahasiswa mampu menjelaskan karakteristik berbagai struktur data serta alur pengelolaan data dalam mendukung penyelesaian permasalahan komputasi.

## Materi Pembelajaran
Double linked list sebagai pengembangan linked list: node dengan dua pointer (prev dan next), operasi insert dan delete pada double linked list.

## Bentuk dan Metode Pembelajaran
Modalitas: Blended Learning. Bentuk: Kuliah. Metode: diskusi kelompok dan studi kasus. Sumber belajar: LMS Polinema.

## Estimasi Waktu
2 x 2 x 100' tatap muka dan tugas terstruktur; 2 x 2 x 70' tugas mandiri; penugasan 1 x 50'.

## Pengalaman Belajar Mahasiswa
Mahasiswa memahami double linked list sebagai pengembangan linked list, menerapkannya dalam diagram flowchart, dan menunjukkan hasil pemahaman.

## Penilaian
Bentuk penilaian: Tugas Double Linked List. Kriteria: rubrik kriteria penilaian; presentasi; tes tulis atau tugas pemecahan studi kasus dengan flowchart.

## Indikator Penilaian
Pemahaman konsep double linked list; ketepatan diagram flowchart sesuai studi kasus; performa presentasi.
