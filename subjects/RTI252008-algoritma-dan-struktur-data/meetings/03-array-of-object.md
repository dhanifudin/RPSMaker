# Pertemuan 3 - Array of Object

## Kemampuan Akhir
- SCPMK903-01701: Mahasiswa mampu menganalisis permasalahan komputasi pada struktur data berbasis array of object, serta menjelaskan dan membandingkan pendekatan algoritmik untuk pengolahan dan pencarian data.

## Materi Pembelajaran
Array of object: deklarasi, instansiasi, assignment, dan penampilan data.

## Bentuk dan Metode Pembelajaran
Modalitas: Blended Learning. Bentuk: Kuliah luring. Metode: diskusi kelompok virtual dan studi kasus. Sumber belajar: LMS Polinema.

## Estimasi Waktu
2 x 2 x 100' tatap muka dan tugas terstruktur; 2 x 2 x 70' tugas mandiri; penugasan 1 x 50'.

## Pengalaman Belajar Mahasiswa
Mahasiswa memahami konsep array of object dan menerapkan array object dalam flowchart untuk berbagai studi kasus.

## Penilaian
Bentuk penilaian: Tugas Array of Object. Kriteria: ketepatan dan penguasaan; keaktifan diskusi; tes tulis atau tugas pemecahan studi kasus dengan flowchart.

## Indikator Penilaian
Pemahaman array of object; ketepatan flowchart sebagai implementasi algoritma untuk menyelesaikan studi kasus.
