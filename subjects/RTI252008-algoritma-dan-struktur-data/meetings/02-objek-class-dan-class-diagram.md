# Pertemuan 2 - Objek, Class, dan Class Diagram

## Kemampuan Akhir
- SCPMK0903-01701: Mahasiswa mampu menganalisis permasalahan komputasi pada struktur data berbasis array of object, serta menjelaskan dan membandingkan pendekatan algoritmik untuk pengolahan dan pencarian data.

## Materi Pembelajaran
Konsep objek, class, atribut, method, instansiasi, konstruktor, dan class diagram.

## Bentuk dan Metode Pembelajaran
Modalitas: Blended Learning. Bentuk: Luring. Metode: diskusi kelompok dan studi kasus. Sumber belajar: LMS Polinema.

## Estimasi Waktu
2 x 2 x 100' tatap muka dan tugas terstruktur; 2 x 2 x 70' tugas mandiri; penugasan 1 x 50'.

## Pengalaman Belajar Mahasiswa
Mahasiswa memahami object, mendeklarasikan class, atribut, method, membuat object, mengakses atribut dan method, serta menuangkan object ke class diagram.

## Penilaian
Bentuk penilaian: Tugas Class Diagram. Kriteria: ketepatan dan penguasaan; keaktifan diskusi; tes tulis atau tugas pemecahan studi kasus.

## Indikator Penilaian
Pemahaman konsep object, class, atribut, dan method; ketepatan pembuatan class diagram.
