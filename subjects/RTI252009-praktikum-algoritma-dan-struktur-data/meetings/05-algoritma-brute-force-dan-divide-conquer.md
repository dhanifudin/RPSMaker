# Pertemuan 05 - Algoritma brute force dan divide-conquer

## Kemampuan Akhir
- SCPMK0903-01801: Mahasiswa mampu menerapkan teknik dasar pemrograman dan algoritma untuk menyelesaikan permasalahan komputasi menggunakan array of object.

## Materi Pembelajaran
Algoritma brute force, divide-conquer, kompleksitas Big-O, dan perhitungan notasi Big-O.

## Bentuk dan Metode Pembelajaran
Modalitas: Blended Learning. Bentuk: Kuliah luring. Metode: diskusi kelompok virtual dan studi kasus. Sumber belajar: LMS Polinema.

1 x 3 x 50' praktikum (asinkron video, sinkron tatap muka, tugas terstruktur); penugasan 1 x 50'.
