# Pertemuan 02 - Konsep objek, class, dan class diagram

## Kemampuan Akhir
- SCPMK0903-01801: Mahasiswa mampu menerapkan teknik dasar pemrograman dan algoritma untuk menyelesaikan permasalahan komputasi menggunakan array of object.

## Materi Pembelajaran
Konsep objek, class, atribut, method, instansiasi, konstruktor, dan class diagram.

## Bentuk dan Metode Pembelajaran
Modalitas: Blended Learning. Bentuk: Luring. Metode: diskusi kelompok dan studi kasus. Sumber belajar: LMS Polinema.

## Estimasi Waktu
1 x 3 x 50' praktikum (asinkron video, sinkron tatap muka, tugas terstruktur); penugasan 1 x 50'.

## Pengalaman Belajar Mahasiswa
Mahasiswa memahami object, mendeklarasikan class, atribut, method, membuat object, mengakses atribut dan method, memanggil konstruktor, serta menuangkan object ke class diagram.

## Penilaian
Rubrik kriteria penilaian; keaktifan diskusi; tes tulis atau tugas pembuatan class diagram.

## Indikator Penilaian
Pemahaman konsep object, class, atribut, dan method; ketepatan pembuatan class diagram.
