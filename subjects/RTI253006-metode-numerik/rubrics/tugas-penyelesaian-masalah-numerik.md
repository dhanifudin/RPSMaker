# Tugas Penyelesaian Masalah Numerik

**Bentuk Tugas/Evaluasi:** Tugas terstruktur individu

**Sub-CPMK:** SCPMK0902-02401, SCPMK0902-02402, SCPMK0902-02403, SCPMK1007-02404

## Deskripsi
Mahasiswa menyelesaikan tugas perhitungan numerik secara individu mencakup analisis galat, penyelesaian sistem persamaan linear, persamaan nonlinear, diferensiasi, integrasi, interpolasi, dan regresi sesuai jadwal yang ditetapkan.

## Metode Pengerjaan
Mempelajari konsep dan algoritma metode numerik, menyelesaikan soal perhitungan step-by-step secara manual dan/atau menggunakan perangkat lunak, mendokumentasikan proses dan hasil, kemudian mengumpulkan sesuai batas waktu.

## Bentuk Format Luaran
Lembar jawaban dengan proses perhitungan lengkap (kerja tangan dan/atau file perangkat lunak), tabel iterasi bila diperlukan, dan analisis galat.
