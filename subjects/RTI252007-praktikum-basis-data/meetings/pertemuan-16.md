# Pertemuan 16 - Studi Kasus 2: Implementasi Basis Data

## Kemampuan Akhir
- SCPMK1001-01601: Mampu mengimplementasikan perancangan skema basis data dan implementasinya melalui query SQL sesuai dengan studi kasus.

## Materi Pembelajaran
Studi Kasus 2: implementasi rancangan ke MySQL menggunakan DDL, pengelolaan data dengan DML, dan query SELECT termasuk multi table.

## Bentuk dan Metode Pembelajaran
Modalitas: Luring. Bentuk: Demonstrasi dan presentasi studi kasus. Metode: Case Method, praktik terbimbing, unjuk kerja.

## Estimasi Waktu
1 x 2 x 50' tatap muka dan praktikum; 1 x 2 x 50' tugas/praktik mandiri.

## Pengalaman Belajar Mahasiswa
Mahasiswa mengimplementasikan rancangan basis data pada MySQL, mengisi dan memanipulasi data, menampilkan data dari beberapa tabel, dan mendokumentasikan laporan implementasi.

## Penilaian
Bentuk penilaian: Case Method Studi Kasus Basis Data. Mengacu rubrik RTM.

## Indikator Penilaian
Ketepatan perancangan; kelengkapan implementasi; kualitas demonstrasi.
