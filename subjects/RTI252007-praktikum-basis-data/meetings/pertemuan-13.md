# Pertemuan 13 - SELECT Multi Table

## Kemampuan Akhir
- SCPMK0701-01601: Mampu mengimplementasikan skema basis data dan mengeksekusi query SQL untuk mengelola data sesuai studi kasus melalui kegiatan praktikum.

## Materi Pembelajaran
SELECT Multi Table: perintah JOIN dan jenisnya (INNER JOIN, LEFT/RIGHT OUTER JOIN, CROSS JOIN) untuk menampilkan data dari beberapa tabel.

## Bentuk dan Metode Pembelajaran
Modalitas: Luring. Bentuk: Praktikum terstruktur. Metode: Case Based Learning (CBL), demonstrasi, dan latihan terbimbing.

## Estimasi Waktu
1 x 2 x 50' tatap muka dan praktikum; 1 x 2 x 50' tugas/praktik mandiri.

## Pengalaman Belajar Mahasiswa
Mahasiswa mengikuti praktikum terbimbing, menuliskan perintah SELECT multi table menggunakan JOIN, menyusun laporan praktikum, dan menunjukkan hasil.

## Penilaian
Bentuk penilaian: Laporan Praktikum Basis Data. Mengacu rubrik RTM.

## Indikator Penilaian
Ketepatan implementasi; kualitas laporan/skrip; demonstrasi hasil.
