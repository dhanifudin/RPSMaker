# Pertemuan 11 - DML MySQL

## Kemampuan Akhir
- SCPMK701-01601: Mampu mengimplementasikan skema basis data dan mengeksekusi query SQL untuk mengelola data sesuai studi kasus melalui kegiatan praktikum.

## Materi Pembelajaran
DML MySQL: pengelolaan data pada basis data menggunakan perintah INSERT, DELETE, UPDATE, dan klausa WHERE.

## Bentuk dan Metode Pembelajaran
Modalitas: Luring. Bentuk: Praktikum terstruktur. Metode: Case Based Learning (CBL), demonstrasi, dan latihan terbimbing.

## Estimasi Waktu
1 x 2 x 50' tatap muka dan praktikum; 1 x 2 x 50' tugas/praktik mandiri.

## Pengalaman Belajar Mahasiswa
Mahasiswa mengikuti praktikum terbimbing, menuliskan perintah DML untuk mengelola data yang tersimpan pada basis data, menyusun laporan praktikum, dan menunjukkan hasil.

## Penilaian
Bentuk penilaian: Laporan Praktikum Basis Data. Mengacu rubrik RTM.

## Indikator Penilaian
Ketepatan implementasi; kualitas laporan/skrip; demonstrasi hasil.
