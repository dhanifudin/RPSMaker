# Pertemuan 12 - Query data

## Kemampuan Akhir
- SCPMK1001-01501: Mampu mengimplementasikan skema basis data dan menyusun query SQL untuk mendefinisikan, memanipulasi, dan menampilkan data sesuai dengan studi kasus.

## Materi Pembelajaran
Query data: proses query data; bahasa SQL-DQL (SELECT, klausa WHERE, ORDER BY, GROUP BY, fungsi agregasi SUM, MIN, MAX, AVG).

## Bentuk dan Metode Pembelajaran
Bentuk: ceramah, diskusi, latihan analisis. Metode: Case Based Learning (CBL). Penugasan: menuliskan perintah SELECT untuk menampilkan kembali data pada basis data.

## Estimasi Waktu
1 x 2 x 50' tatap muka dan tugas terstruktur; 1 x 2 x 50' tugas mandiri.

## Pengalaman Belajar Mahasiswa
Mahasiswa menjelaskan proses query data serta menuliskan perintah SQL-DQL untuk menampilkan kembali data yang disimpan dalam basis data.

## Penilaian
Kriteria: ketepatan dan penguasaan. Bentuk: keaktifan diskusi kelompok; ketepatan jawaban tugas.

## Indikator Penilaian
Ketepatan perintah SQL-DQL untuk menampilkan kembali data sesuai studi kasus.
