# Pertemuan 06 - Praktik pemetaan ERD hasil rancangan ke model relasional

## Kemampuan Akhir
- SCPMK0203-01501: Mampu memahami konsep data, basis data, dan basis data relasional, serta merancang model dan skema basis data secara konseptual dan logikal dengan memperhatikan integritas data dan konsep normalisasi.

## Materi Pembelajaran
Praktik pemetaan ERD hasil rancangan ke model relasional dan penilaian kesesuaian model relasional dengan requirement data.

## Bentuk dan Metode Pembelajaran
Bentuk: ceramah, diskusi, latihan analisis. Metode: Case Based Learning (CBL). Penugasan: Tugas 5 memetakan ERD rancangan sendiri ke model relasional.

## Estimasi Waktu
1 x 2 x 50' tatap muka dan tugas terstruktur; 1 x 2 x 50' tugas mandiri.

## Pengalaman Belajar Mahasiswa
Mahasiswa memetakan ERD hasil rancangan sebelumnya ke model relasional dan menilai kesesuaiannya dengan requirement data.

## Penilaian
Kriteria: ketepatan dan penguasaan. Bentuk: keaktifan diskusi kelompok; ketepatan jawaban tugas.

## Indikator Penilaian
Ketepatan model relasional dan kesesuaiannya dengan requirement data.
