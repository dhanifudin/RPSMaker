# Pertemuan 11 - Pengelolaan data pada basis data

## Kemampuan Akhir
- SCPMK1001-01501: Mampu mengimplementasikan skema basis data dan menyusun query SQL untuk mendefinisikan, memanipulasi, dan menampilkan data sesuai dengan studi kasus.

## Materi Pembelajaran
Pengelolaan data pada basis data: penambahan, penghapusan, perubahan data; bahasa SQL-DML (INSERT, DELETE, UPDATE, klausa WHERE).

## Bentuk dan Metode Pembelajaran
Bentuk: ceramah, diskusi, latihan analisis. Metode: Case Based Learning (CBL). Penugasan: Tugas 9 menuliskan perintah DML untuk mengelola data yang tersimpan pada basis data.

## Estimasi Waktu
1 x 2 x 50' tatap muka dan tugas terstruktur; 1 x 2 x 50' tugas mandiri.

## Pengalaman Belajar Mahasiswa
Mahasiswa menjelaskan proses pengelolaan data serta menuliskan perintah SQL-DML untuk menambah, menghapus, dan mengubah data yang tersimpan pada basis data.

## Penilaian
Kriteria: ketepatan dan penguasaan. Bentuk: keaktifan diskusi kelompok; ketepatan jawaban tugas.

## Indikator Penilaian
Ketepatan perintah SQL-DML untuk menambah, menghapus, dan mengubah data sesuai studi kasus.
