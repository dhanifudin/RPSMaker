# Pertemuan 13 - Perintah SELECT untuk multi table

## Kemampuan Akhir
- SCPMK1001-01501: Mampu mengimplementasikan skema basis data dan menyusun query SQL untuk mendefinisikan, memanipulasi, dan menampilkan data sesuai dengan studi kasus.

## Materi Pembelajaran
Perintah SELECT untuk multi table: perintah JOIN dan jenisnya (INNER JOIN, OUTER JOIN, CROSS JOIN).

## Bentuk dan Metode Pembelajaran
Bentuk: ceramah, diskusi, latihan analisis. Metode: Case Based Learning (CBL). Penugasan: menuliskan perintah SELECT multi table untuk menampilkan data dari beberapa tabel.

## Estimasi Waktu
1 x 2 x 50' tatap muka dan tugas terstruktur; 1 x 2 x 50' tugas mandiri.

## Pengalaman Belajar Mahasiswa
Mahasiswa menjelaskan perintah SELECT untuk multi table beserta jenisnya dan menuliskan perintah SELECT multi tabel sesuai kebutuhan.

## Penilaian
Kriteria: ketepatan dan penguasaan. Bentuk: keaktifan diskusi kelompok; ketepatan jawaban tugas.

## Indikator Penilaian
Ketepatan perintah SELECT untuk menampilkan data pada multi table sesuai studi kasus.
