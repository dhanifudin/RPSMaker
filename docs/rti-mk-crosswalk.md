# RTI25 → MK → CPL/CPMK Crosswalk

Semua 59 mata kuliah kini memiliki RPS yang sudah di-author. Kolom CPL dan
CPMK utama pada tabel di bawah diekstrak langsung dari CP block dan CPMK
Induk pada `<Nama MK> - RPS.tex` masing-masing mata kuliah (bukan lagi hasil
reverse-engineering/desain awal) — lihat juga
`docs/cpl-cpmk-subcpmk-mk-grading-map.md` untuk detail Sub-CPMK dan
penilaiannya per mata kuliah.

MK numbering: sequential per kurikulum order (Sem 1 = MK001–009, Sem 2 = MK010–018, dst.).
Sem 6 reguler = MK048–052; Sem 6 magang = MK053–056; Sem 7 = MK057; Sem 8 = MK058–059.

## Sub-CPMK Numbering Convention

Format: `SCPMK<cpmk>-<MKnum3digit><seq2digit>`

Contoh: SCPMK0211-02001 = Sub-CPMK dari CPMK0211, MK020 (RTI253002 SIM), indikator 01.

Per course, sequencing is: 01, 02, 03, ... regardless of which CPMK each sub-CPMK belongs to.
